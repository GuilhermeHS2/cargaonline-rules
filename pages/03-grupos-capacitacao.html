SEÇÃO 3

Grupos de Capacitação

Organizam usuários dentro de uma transportadora, controlando o acesso a treinamentos, comunicados e funcionalidades.

3.1 Regras

  • Pertence a uma única transportadora.
  • Pode ter número máximo de vagas. Quando atingido, novos usuários não entram.
  • Pode ser ativado ou desativado — usuários de grupos desativados perdem o acesso ao conteúdo.
  • Um usuário pode participar de vários grupos ao mesmo tempo.
  • Grupos excluídos ficam arquivados — o histórico é preservado.
  • Pode ter datas de ativação e desativação automática.
  • Pode ter horários de acesso por dia da semana — fora do horário configurado, o acesso ao grupo é bloqueado.
  • É possível gerar um link de convite com limite de usos para entrada no grupo.

3.2 Comunidade

  • O Admin Master pode habilitar um feed de publicações dentro do grupo.
  • Fotos publicadas são apagadas automaticamente após 90 dias.
  • Moderação pode ser ativada — publicações precisam de aprovação antes de aparecer para todos.
